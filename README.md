## What I Have Built

**Privacy Policy Analyzer** — A Chrome extension that lets you select any text on a webpage, right-click, and get an instant ML-powered risk score. Backed by a FastAPI backend trained on 15,507 annotated segments at 89% accuracy.

**CoachAI** — A sports performance analysis tool that compares a coach reference video with a player recording using pose estimation and edge detection. Also has a live mode that overlays skeletal keypoints directly on the player camera feed in real time.

**Basha AI** — A multilingual chatbot supporting 5 languages with automatic language detection. Built with FastAPI and React using server-sent events for low-latency streaming.

**AI Learning Path Assistant** — An AI agent built with the Gemini API and Model Context Protocol that generates personalized learning roadmaps and automates YouTube playlist creation via Pipedream.

**Serverless Image Pipeline** — An event-driven AWS Lambda pipeline that processes images on S3 upload. Load tested at 120 concurrent uploads with zero failures.
